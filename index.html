常见问题
❓ 如何使用NeoXe OS？
方法1（U盘启动）： 使用 Ventoy 或 Rufus 将 ISO 写入 U盘，从 U盘启动即可。
方法2（本地硬盘引导）： 使用 EasyBCD（Windows）或 BOOTICE 添加 NeoXe OS 的 WIM 启动项。步骤：① 将 ISO 中的 boot.wim / 系统文件解压到本地分区（如 D:\NeoXe）；② 打开 EasyBCD → 添加新条目 → WinPE → 路径选择 D:\NeoXe\boot.wim → 添加；③ 重启选择 NeoXe OS 进入。
注意： 系统运行于内存，重启后更改丢失，适合维护/体验。
❓ 硬件要求？
CPU： 建议 Intel Nehalem 架构或更高 (64位)，AMD 推土机架构或更高。24H2版本需要 SSE4.2。
内存： SuperLite 最低 1GB，Slim 推荐 2GB，Max 推荐 4GB，Elite 推荐 6GB+，Ultra 推荐 8GB+。
SuperLite 可在单核 CPU + 1GB 内存下运行。支持 BIOS / UEFI。
❓ 如何联网/装驱动？
Elite/Ultra内置完整网络驱动，WiFi可直接连接。特殊硬件需手动注入驱动。
❓ 为什么软件无法运行？
缺少VC++/.NET运行库。Elite/Ultra已集成.NET，其他系列请自行安装或使用便携版。
❓ 能保存文件吗？
重启后数据清零，重要文件请存至U盘/其他硬盘分区。持久化请使用RAMOS或WTG。
❓ 最新版本去哪获取？
关注B站 XeRiver 或 QQ群 1075906894，GitHub及国内网盘同步更新。
